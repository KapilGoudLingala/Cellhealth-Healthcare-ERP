## System Architecture

The system is built on the Frappe Framework using ERPNext Healthcare as the base.

### Key Components
- Patient Appointment
- Patient Encounter (Core Workflow Document)
- Lab Test & Service Requests
- Sales Invoice & Payment Entry
- Medication Requests
- Nutrition Assessment

### Architecture Principles
- Single source of truth (Patient Encounter)
- Workflow-controlled transitions
- Role-based permissions
- Audit-friendly design
