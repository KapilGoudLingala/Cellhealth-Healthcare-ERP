# CellHealth – Healthcare ERP Platform

## Overview
CellHealth is a production-grade Healthcare ERP platform built using the **Frappe Framework and ERPNext Healthcare**, implemented for a live Cell Health Clinic.

The system manages the **complete patient lifecycle** through a strictly controlled, role-based workflow covering reception, counselling, doctors, laboratory, billing, pharmacy, and nutrition departments.

This repository documents the **system design, workflows, portals, and core business logic**.

> ⚠️ Source code is not included due to client access and data privacy restrictions.

---

## Key Objectives
- Standardize patient flow across departments
- Prevent billing and prescription mismatches
- Enable longitudinal lab analysis
- Ensure accountability via workflow enforcement
- Improve operational efficiency

---

## Technology Stack
- Frappe Framework
- ERPNext Healthcare
- Python & JavaScript
- MariaDB
- Frappe Workflow Engine
- Script & Query Reports

---

## Core Workflow
The **Patient Encounter** document acts as the backbone of the system and moves through **18+ workflow states**, ensuring no clinical or billing step is skipped.
